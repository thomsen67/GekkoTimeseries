Signatures/checksums

At the moment, Gekko uses SHA1 checksums. You may download a Gekko version and drag & drop the downloaded file onto this online SHA1 page page. You may alternatively use the Microsoft command line tools Sigcheck (with -h option) or fciv (with -sha1 option). If you get the same checksum, the file has not been tampered with since it was compiled.
